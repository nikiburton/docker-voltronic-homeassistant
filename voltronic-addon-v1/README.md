# Voltronic Inverter MQTT Addon

Addon para Home Assistant que lee datos del inversor Voltronic y publica en MQTT.

## Instalación

1. Añadir repositorio en Home Assistant: 
https://github.com/TU_USUARIO/voltronic-addon

markdown
Copy code
2. Instalar y configurar el addon.
3. Configurar MQTT y dispositivo (`/dev/hidraw0`) en la UI.
